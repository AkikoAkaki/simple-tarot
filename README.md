# Simple Tarot

A minimalist, high-end Tarot web application built with React, Vite, Framer Motion, and Tailwind CSS.
The project has recently been migrated to **TypeScript** for better type safety and developer experience.

## Features
- **Tarot Spreads**: Draw cards using various predefined layouts (Single Card, Three Cards, Celtic Cross).
- **Responsive Animations**: Smooth flip interactions and elegant layout transitions.
- **Card Meanings**: Detailed upright and reversed meanings for all 78 standard Tarot cards.
- **Offline Generation**: The deck data is generated dynamically via a TypeScript offline script (`generate_cards.ts`).

## Tech Stack
- **Core**: React 19, TypeScript
- **Build**: Vite
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion

## Development
```bash
npm install
npm run dev
```
